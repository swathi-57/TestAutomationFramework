# 🚀 Test Automation Framework for REST APIs

A scalable and modular **API Test Automation Framework** built using **Java, Rest Assured, JUnit, and Mockito** to automate REST API testing with high reliability and maintainability.

---

## 📌 Overview

This project demonstrates the design and implementation of a robust automation framework for testing RESTful APIs. It focuses on **test case automation, validation, debugging, and system reliability**, aligning with real-world industry practices.

---

## 🛠️ Tech Stack

- Java
- Rest Assured
- JUnit 5
- Mockito
- Maven
- Jackson (JSON Processing)

---

## ✨ Key Features

✔️ Automated API testing (GET, POST, authentication)  
✔️ Reusable test setup using Base Test class  
✔️ Config-driven environment management  
✔️ JSON serialization/deserialization  
✔️ Mocking dependencies using Mockito  
✔️ Clean and modular project structure  
✔️ Scalable for large test suites  

---

## 📂 Project Structure

```text

test-automation-framework/
│── src/
│ ├── main/java/utils/ # Utility classes
│ ├── test/java/base/ # Base test setup
│ ├── test/java/tests/ # Test cases
│ ├── test/java/models/ # Request/Response models
│ └── resources/ # Config files
│── pom.xml
```

## 🧪 Test Scenarios Covered

- ✅ Fetch users (GET API)
- ✅ Create user (POST API)
- ✅ Authentication success & failure cases
- ✅ Response validation (status code, body)
- ✅ Edge case handling

## ▶️ How to Run the Project

### Prerequisites
- Java 8+
- Maven installed

### Steps
```text
git clone https://github.com/YOUR_USERNAME/test-automation-framework.git
cd test-automation-framework
mvn clean test
```
